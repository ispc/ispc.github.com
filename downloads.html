Downloading The Intel® Implicit SPMD Program Compiler

The Intel® Implicit SPMD Program Compiler (Intel® ISPC) is available in both source and binary form.

Here are latest released ISPC binaries based on LLVM 18.1.8:

  • Windows (64 bit) ispc binary (v1.25.3)
  • macOS (x86, 64 bit, 10.12 Sierra) ispc binary (v1.25.3)
  • macOS (ARM64, 11.0 Big Sur) ispc binary (v1.25.3)
  • macOS (Universal Binary, 10.12 Sierra) ispc binary (v1.25.3)
  • Linux (64 bit) ispc binary (v1.25.3)
  • Linux (64 bit) ispc binary for OneAPI (v1.25.3)
  • Linux (64 bit) ispc binary for AARCH64 (v1.25.3)
  • Examples packed in tar.gz archive (v1.25.3)
  • Examples packed in zip archive (v1.25.3)

All archives are signed with a following PGP key.

Here are alternative ways to download and install ISPC:

  • Intel® oneAPI Distribution
  • Linux (Snap Store)

Here is installation instruction for Windows:

  • Windows installation instruction

Here are latest ISPC binaries corresponding to main branch (based on LLVM 17.0.6):

  • Windows (64 bit) ispc binary (v1.26.0dev)
  • Linux (64 bit) ispc binary (v1.26.0dev)

Here is LLVM 3.5 based binary with experimental NVPTX support:

  • Linux (64 bit) ispc binary and examples with NVPTX support (v1.8.2)

After installation, add the platform-appropriate binary executable to your PATH. Then see the ispc user's guide and the examples in the examples/ directory of the distribution.

You can download previous versions on Github releases page.

Alternatively, source can be downloaded. If you're using git, run:

% git clone https://github.com/ispc/ispc.git ispc

Finally, zip and tar files of the source can be downloaded:

  • zip file of the top-of-tree source.
  • tar.gz file of the top-of-tree source.
  • zip file of the v1.25.3 source, corresponding to the binaries above.
  • tar.gz file of the v1.25.3 source, corresponding to the binaries above.

There is a page in the ispc wiki with instructions about building from source on Windows, Linux and macOS.
