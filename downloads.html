Downloading The Intel® SPMD Program Compiler

The Intel® SPMD Program Compiler (ispc) is available in both source and binary form.

Here are latest released ISPC binaries based on LLVM 8.0:

  • Windows (32 bit) ispc binary and examples (v1.12.0)
  • macOS (64 bit, Sierra) ispc binary and examples (v1.12.0)
  • Linux (64 bit) ispc binary and examples (v1.12.0)

Here are latest ISPC binaries corresponding to master branch (based on LLVM 10.0):

  • Windows (64 bit) ispc binary and examples (v1.12.1dev)
  • Linux (64 bit) ispc binary and examples (v1.12.1dev)

Here is LLVM 3.5 based binary with experimental NVPTX support:

  • Linux (64 bit) ispc binary and examples with NVPTX support (v1.8.2)

To install the compiler, install the platform-appropriate binary executable in your PATH. Then see the ispc user's guide and the examples in the examples/ directory of the distribution.

You can download previous versions on Github releases page.

Alternatively, source can be downloaded. If you're using git, run:

% git clone https://github.com/ispc/ispc.git ispc

Finally, zip and tar files of the source can be downloaded:

  • zip file of the top-of-tree source.
  • tar.gz file of the top-of-tree source.
  • zip file of the v1.12.0 source, corresponding to the binaries above.
  • tar.gz file of the v1.12.0 source, corresponding to the binaries above.

There is a page in the ispc wiki with instructions about building from source on Windows, Linux and macOS.
