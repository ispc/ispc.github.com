250 /** This function takes chains of InsertElement instructions along the
251  lines of:
252 
253  %v0 = insertelement undef, value_0, i32 index_0
254  %v1 = insertelement %v1, value_1, i32 index_1
255  ...
256  %vn = insertelement %vn-1, value_n-1, i32 index_n-1
257 
258  and initializes the provided elements array such that the i'th
259  llvm::Value * in the array is the element that was inserted into the
260  i'th element of the vector.
261 
262  When the chain of insertelement instruction comes to an end, the only
263  base case that this function handles is the initial value being a
264  constant vector. For anything more complex (e.g. some other arbitrary
265  value, it doesn't try to extract element values into the returned
266  array.
267 
268  This also handles common broadcast pattern:
269  %broadcast_init.0 = insertelement <4 x i32> undef, i32 %val, i32 0
270  %broadcast.1 = shufflevector <4 x i32> %smear.0, <4 x i32> undef,
271  <4 x i32> zeroinitializer
272  Function returns:
273  Compare all elements and return one of them if all are equal, otherwise NULL.
274  If compare argument is false, don't do compare and return first element instead.
275  If undef argument is true, ignore undef elements (but all undef yields NULL anyway).
276  */
277 extern llvm::Value * LLVMFlattenInsertChain (llvm::Value *inst, int vectorWidth,
278  bool compare = true, bool undef = true);
